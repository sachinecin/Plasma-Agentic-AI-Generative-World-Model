# Examples

This directory contains example scripts and demonstrations of the Plasma framework for logistics and other use cases.

## Available Examples

Examples will demonstrate:

- **Logistics Optimization**: Supply chain and routing examples
- **Task Planning**: Multi-agent coordination scenarios
- **Real-time Adaptation**: Edge deployment demonstrations
- **Visual Learning**: Examples using visual error feedback
- **Adversarial Testing**: Reward hacking prevention demos

## Running Examples

Each example is self-contained and can be run independently:

```bash
python examples/logistics_optimization.py
python examples/phantom_path_demo.py
```

## Contributing

When adding new examples:
1. Include clear documentation and comments
2. Minimize external dependencies
3. Provide expected output or screenshots
4. Add requirements.txt if additional packages are needed
